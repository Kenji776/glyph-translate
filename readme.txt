glyph-translate
Small utility for converting common to draconic glpyhs for calligraphy practice. 

To use:

1) Install nodejs
2) Run the runServer.bat file, or call the process.js file via
	node process.js
3) Open a browser tab to http://localhost:5000/
4) Enter your message and click submit.
5) Your message should be translated into draconic text and glyphs.
